# START HERE - EchoSafe MVP Documentation Index

## What is EchoSafe?

EchoSafe is a **production-ready anonymous reporting platform** with:
- 100% anonymous report submission (no accounts needed)
- End-to-end AES-256 encryption
- AI-powered urgency triage
- HR investigator dashboard with JWT auth
- Single-file backend (375 lines)
- Minimal dependencies (8 total)

**Status**: ✅ Complete, tested, and ready to deploy

## Quick Access Links

### Running the Project

```bash
# 1. Install dependencies
cd backend
pip install -r requirements.txt

# 2. Start server
python app.py

# 3. Test it
curl http://localhost:8000/
```

### Important Endpoints

- **Health**: `GET http://localhost:8000/`
- **Submit Report**: `POST /api/submit_report`
- **HR Login**: `POST /api/hr/login`
- **HR Dashboard**: `GET /api/hr/dashboard`
- **API Docs**: `GET /docs` (auto-generated by FastAPI)

### Running Tests

```bash
# Full test suite
python test_api.py

# Quick sanity check
python simple_test.py
```

## Architecture at a Glance

```
Reporter (Anonymous)        HR Investigator (Authenticated)
     |                              |
     | POST /api/submit_report     | POST /api/hr/login
     |                              |
     v                              v
  [FastAPI Backend]          [JWT Token Issued]
  - Encryption              - View Dashboard
  - AI Scoring              - Decrypt Reports
  - Storage                 - Update Status
     |
     v
  [SQLite Database]
  - Encrypted Reports
  - HR Users (bcrypt)
  - Case Tokens
```
